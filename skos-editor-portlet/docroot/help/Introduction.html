Introduction


"SKOS Editor" is a standalone deployable portlet plugin application for Liferay portal. Current plugin run on the most recent stabil Liferay Portal 6.2.0 version.
Portlet is written according to JSR 286 specification. Plugins backend is done in Java, project is compiled under JDK 1.7 version.

Plugins architectute follows a typical MVC pattern, where (M)odel is using Liferays ServiceBuilder feature to handle the database communication through Hibernate.

The database is MySQL 5.6, plugin tables are automatically created by the ServiceBuilder and are of InnoDB type. Portlet is using the database to save files and file
versions, as well as user specific settings, and collaboration details and history.

The (V)iew part of MVC pattern is implemented as JSP pages and is actively using JSTL tags. The hierarchy tree is implemented in pure JSP, without any third-party
libraries. Client-side functionality is supported by the latest jQuery library and custom JavaScript code.

The (C)ontroller part is based on Liferays default GenericPortlet implementation.

The plugin is an SPA (Single Page Application). Once loaded, all the further processing is performed on the same page without any reloads. Server and client
communication is based on Ajax calls using jQuery library from the client-side and Liferay portlet from the server-side.
