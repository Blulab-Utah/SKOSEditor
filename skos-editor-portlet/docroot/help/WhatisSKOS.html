What is SKOS?


SKOS is a W3C standard designed to support the development of knowledge organization systems (KOS) such as thesauri, classification schemes, subject heading systems and taxonomies within the framework of the Semantic Web. SKOS provides a standard way to represent knowledge organization systems using the Resource Description Framework (RDF). Encoding this information in RDF allows it to be passed between computer applications in an interoperable way.  SKOS-related development work is organized  via the public-esw-thes@w3.org mailing list.  A SKOS community wiki is also provided as a support for discussing and exchanging datasets, tools, and tutorial material.
